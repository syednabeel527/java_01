SOME BASIC COMMANDS TO USE 
step by step
1. git config --global user.name "username of git account"
2. git config --global user.email "your email which you used to create the git account"
3. git config --list # this command is user to verify your user name and email
4. git clone "add the https link from your repository"
5. git status # this command is used to check status
6. git add "file name" # this command is used to add the new or changed file
7. git add . # this command is used to add all files
8. git commit -m "give a meaniful commit name here"
9. git push origin main #this command is used to upload local repo to remote repo
